find-asksci

This app aims to find the redditor in /r/askscience best qualified to address a technical question. The choice is based on similarity between then question and the total corpus of redditors' comments. (HINT: Longer questions with more technical terminology typically yield the best results.)

Please visit my website or the find-asksci github repository for code and discussion.
